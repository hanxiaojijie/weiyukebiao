# 未雨课表缺陷提报模板

测试部门提问题时，建议统一按下面格式提交，方便研发直接复现和定位。

## 一、缺陷提报模板

```text
【问题标题】

【问题等级】
阻断 / 严重 / 一般 / 建议

【所属模块】
首页 / 课程 / 课表 / 打卡 / 奖励 / 统计 / 个人中心 / 其他

【测试环境】
- 设备型号：
- 操作系统版本：
- 微信版本：
- 网络环境：

【测试账号】
账号 A / 账号 B / 其他

【前置条件】

【复现步骤】
1.
2.
3.

【实际结果】

【期望结果】

【复现频率】
必现 / 高概率 / 低概率 / 仅出现一次

【附件】
截图 / 录屏 / 日志
```

## 二、缺陷等级建议

### 阻断

- 无法进入小程序
- 核心页面打不开
- 核心主链路完全无法执行

### 严重

- 能进入页面，但关键结果错误
- 打卡、学分、奖励、统计结果明显不正确
- 用户数据串号

### 一般

- 局部逻辑异常
- 页面展示异常
- 文案、交互、小范围错误

### 建议

- 样式优化
- 文案优化
- 体验改进建议
